Frontend Tech Stack:
- React
- TypeScript
- Tailwind CSS
- Lucide React
- React Router DOM

Backend Tech Stack:
- Django
- Django Rest Framework
- PostgreSQL
- Redis
- Docker

Mobile Tech Stack:
- Flutter
- Dart
- Firebase + google auth
- fvm (flutter version manager)

## Frontend Rules
- Always use TypeScript for frontend development.
- Use Tailwind CSS for styling.
- Use Lucide React for icons.
- Use React Router DOM for routing.
- use manage.sh for memanage project (always check using manage.sh status for project status before running any command)
